# 분류_Classification

> **Machine Learning의 Predictive Model**
>
> - y가 숫자 : 수치예측 ->> Regression `(y = wx + b)`
> - y가 문자 : 범주예측(분류) ->> Classification(Logistic Regression)

> sigmoid() ==> logistic 와 유사한 개념으로 이해
>
> sigmoid 함수 결과값을 0.5를 기준으로 0과 1로 구분

  - 예) 성별, 구매/비구매, 양성/음성, 정상/불량
